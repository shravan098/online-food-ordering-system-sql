# 🍔 Online Food Ordering System (SQL Case Study)

## 📖 Overview

This project demonstrates the design and implementation of a relational database for an Online Food Ordering System using SQL.

The database manages customers, restaurants, menu items, orders, order details, and customer reviews while enabling business analysis through SQL queries.

---

## 🎯 Project Objectives

- Design a normalized relational database
- Implement relationships using primary and foreign keys
- Manage customer and restaurant data
- Track food orders
- Store customer reviews
- Generate business insights using SQL

---

## 🗄️ Database Tables

- Customers
- Restaurants
- MenuItems
- Orders
- OrderDetails
- Reviews

---

## 📊 Business Analysis

This project includes SQL queries to analyze:
